[Notes]

1. Configuring the BIOS requires an Amstrad system disk. A program called NVR.COM is needed.
2. Uses a Paradise Systems EGA card internally. Unlike the PC1512, other video cards can be used.
3. Unlike the PC1512, CPU speed is configurable. However, RAM size cannot be changed.
4. Comes with it's own Amstrad mouse
