[8088] Compaq Portable

[History]

The Compaq Portable was one of the first IBM PC clones, as well as one of the first 'portable' computers.
The machine uses a unique CGA card, and will not display any output with the IBM cards.

[Notes]

1. Not all versions of DOS 2 support hard disks. Using PC-DOS 2.0 or later is recommended.
2. The BIOS is not configurable by the user.
3. If booted without a floppy in the drive, the system will attempt to boot from hard disk
4. To check available memory, run CHKDSK in DOS.
5. Windows 2.1 386 will not work. Windows 3.1 onwards requires a 286 and 2 MB of RAM.
6. Use the Compaq CGA cards with this machine. IBM CGA cards will not work. Disable Snow Emulation for the card, to prevent noise when scrolling through text.
